# Font to Image

Turn (oft / ttf) file to image

## Install

* python 3.5+
* Package: Pillow, Fonttools

```bash
pip install pillow fonttools
```

```bash
git clone 
```

## Usage

```bash
python ttf_2_img.py otf_file
```

## args

* `--image_folder` - where to put image folder
* `--image_size` - size of image
* `--font_size` - size of font
